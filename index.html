Fig. 04

AI Video Recipe Extraction

Python · Gemini AI · SQLAlchemy

This project is an AI-powered recipe extraction and database management system that automatically analyzes cooking videos and converts them into structured recipe data. Using Google's Gemini AI model, the system extracts the recipe name, description, preparation time, cooking instructions, and ingredients from a video. The extracted information is then stored in a SQLite database using SQLAlchemy. To avoid duplicate ingredient entries, the project uses fuzzy string matching to compare AI-generated ingredient names with existing database records and links them based on similarity. This automation reduces manual data entry and enables efficient creation of a searchable, well-organized recipe database from cooking videos.

QueryDoc

Project Overview

Generative AI is powerful, but it hallucinates. When accuracy is paramount—whether for academic research, legal discovery, or corporate knowledge bases—relying on a model's pre-trained memory isn't enough.

QueryDoc bridges this gap. By leveraging Retrieval-Augmented Generation (RAG), it allows users to chat directly with their own documents. Instead of generic answers, QueryDoc extracts precise information grounded in the uploaded text, providing transparent source citations for every claim. It's the speed of an LLM with the reliability of a reference library.
