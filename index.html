AI-Powered Industrial Vision Inspection System

View on GitHub

System Demonstration

AI-Powered Industrial Vision Inspection System is a real-time machine vision simulation that replicates an industrial conveyor belt inspection station used in manufacturing and packaging industries. The system combines Unity for the digital twin simulation, FastAPI for backend communication, and OpenCV for computer vision-based inspection.

Packages travel along a conveyor belt where an industrial inspection camera captures images using Unity's RenderTexture system. These images are sent to a Python FastAPI server, where OpenCV analyzes the package and classifies it as PASS or FAIL based on visual inspection.

Packages that fail inspection are automatically rejected from the conveyor using a simulated reject mechanism, while accepted packages continue through the production line.

Architecture

Package Spawn ↓ Conveyor Belt ↓ Industrial Inspection Camera ↓ RenderTexture Image Capture ↓ FastAPI Server ↓ OpenCV Image Processing ↓ PASS / FAIL Classification ↓ Inspection Result Sent Back to Unity ↓ Status Indicator ↓ Automatic Reject Mechanism

Key Features

  • Real-time conveyor belt simulation
  • Industrial inspection camera simulation
  • Image capture using Unity RenderTexture
  • Computer vision inspection using OpenCV
  • FastAPI REST API integration
  • Real-time Unity ↔ Python communication
  • Automatic PASS / FAIL classification
  • Automated reject mechanism for defective products
  • Live inspection status visualization
  • Modular architecture for future AI model integration
